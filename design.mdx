## Brand Colors

The WRLD Tech brand uses a modern blue palette:

- **Primary**: #2563EB
- **Light**: #3B82F6
- **Dark**: #1D4ED8

## Typography

We use clean, modern fonts for readability across all platforms.

## Logo Usage

WRLD logos should be used consistently across all materials. See the logo assets in the `/logo` directory for light and dark variants.

## Design Principles

1. **Clarity** - Information should be easy to find and understand
2. **Consistency** - Maintain consistent patterns across products
3. **Accessibility** - Design for all users
4. **Performance** - Prioritize fast, responsive experiences
